# NatashaV2

🚀 **NatashaV2** is an evolving AI-powered assistant. It currently features vision-based hand gesture detection integrated with Spotify, enabling touchless media control. But this is just the beginning. This passion project is designed to grow into a multimodal AI system with capabilities in mood detection, language processing, data mining, content generation, and beyond.

## 🌟 Features

### ✅ **Current Capabilities**
- **Hand Gesture Detection**: Uses computer vision to track hand movements and interpret gestures.
- **Spotify Integration**: Hands-free control over playback, including:
  - Shuffle
  - Next/Previous track
  - Volume adjustments

### 🔮 **Future Roadmap**
- **Mood Detection**: AI-driven emotion recognition to personalize experiences.
- **Language Processing**: Natural language understanding for deeper interactions.
- **Data Mining & Analysis**: Extracting insights from structured and unstructured data.
- **Content Generation**: AI-powered research and creative output.

## 🛠️ Installation
To get started with NatashaV2, clone the repo and install dependencies:
```bash
 git clone https://github.com/abhinav-pandey29/NatashaV2.git
 cd NatashaV2
 python -m venv env
 env\Scripts\activate 
 pip install -r requirements.txt
```

## 🚀 Usage
Run the vision-based controller with:
```bash
python vision_spotify_controller.py
```
Make sure your webcam is enabled and Spotify is running.

Run tests with:
```bash
pytest tests\
```
